# Image Processing Assignment - OpenCV

This project demonstrates basic image processing techniques using OpenCV in Python.

## Features

* Image loading and RGB conversion
* Channel splitting (R, G, B)
* Grayscale and thresholding
* Image cropping
* Adding text (branding)
* Image resizing and interpolation comparison

## Tools Used

* Python
* OpenCV
* Matplotlib
* Google Colab

## Note

The image used in this project was captured personally as required.
